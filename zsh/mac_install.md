# 因为被墙的原因只能选择手动安装

## clone文件
git clone https://github.com/ohmyzsh/ohmyzsh.git ~/.oh-my-zshh

cp ~/.oh-my-zsh/templates/zshrc.zsh-template ~/.zshrc

chsh -s $(which zsh)

## 安装后颜色不对
安装iTerm2

安装solarized

安装Meslo字体
